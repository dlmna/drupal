Setup
=====

Install Composer
----------------
sudo apt-get update
sudo apt-get install curl php5-cli git
curl -sS https://getcomposer.org/installer | sudo php -- --install- dir=/usr/local/bin --filename=composer

Configure Apache
----------------
<VirtualHost *:80>
    ServerName 192.168.1.223
    ServerAlias drupal.local
    DocumentRoot /var/www/drupal
</VirtualHost>

Create MySql database and user
------------------------------
drupal

Update /etc/hosts, add
----------------------
127.0.0.1   drupal.local

Get project via git
-------------------
git clone --branch 8.1.8-dev git@github.com:dlmna/drupal.git drupal
cd drupal
composer install

Make Drush Public
-----------------
sudo ln -s ~/workspace/mgepa/drupal/vendor/drush/drush/drush /usr/local/bin/drush

Add Symlink
-----------
cd /var/www
sudo ln -s ~/workspace/mgepa/drupal drupal

Goto drupal.local and follow instructions
-----------------------------------------
choose standard installation


Configuration
=============

after installation
goto Erweitern
install extensions
 * Workbench
 * Workbench moderation
 * Field collection
 * Configuration Translation
 * Content Translation
 * HAL
 * HTTP Basic Authentication
 * RESTful Web Services
 * Serialization
 * REST UI
 * config_devel

goto /admin/config/regional/content-language
configure translation for content types and field collections

goto /admin/structure/types/manage/article/moderation
enable moderation for content type


Debugging
=========

with intellij:
configure xdebug
edit configurations
add new PHP Web Application
add new Server
click Start listening for PHP Debug Connections
