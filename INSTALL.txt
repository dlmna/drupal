install composer
run composer install

goto localhost
follow instructions (Standardinstallation)

after installation
goto Erweitern
install extensions
 * Workbench
 * Workbench moderation
 * Field collection
 * Configuration Translation
 * Content Translation
 * HAL
 * HTTP Basic Authentication
 * RESTful Web Services
 * Serialization
 * REST UI


Configuration
=============

goto /admin/config/regional/content-language
configure translation for content types and field collections

goto /admin/structure/types/manage/article/moderation
enable moderation for content type


Debugging
=========

with intellij:
configure xdebug
edit configurations
add new PHP Web Application
add new Server
click Start listening for PHP Debug Connections
